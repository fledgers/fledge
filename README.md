# Pathfinders 🧭

A centralized platform for NUS students to discover opportunities, track deadlines, and explore modules — all in one place.

## The Problem
As NUS freshmen, we found that internships, hackathons, research programmes, and networking events are scattered across faculty websites, emails, Telegram groups, and social media. Valuable opportunities get missed. Deadlines sneak up. And there's no way to filter what's actually relevant to your year of study or interests.

Pathfinders fixes that.

## What It Does
- 🔍 **Discover** opportunities — internships, competitions, research, exchange programmes, leadership roles
- 🎯 **Filter** by category, year of study, and area of interest
- 📚 **Explore modules** based on skills you want to develop
- 📅 **Track deadlines** so nothing slips through
- 🔖 **Bookmark** opportunities to revisit later
- ⭐ **Get recommendations** based on your profile

## Tech Stack
- React + Tailwind CSS (frontend)
- JavaScript
- NUSMods API (module data)
- Supabase (database)
- Vercel (deployment)
